## 🔧 技术执行细节

### 自动化工具
- **工具名称**: CSDN-Publisher
- **版本**: v1.0.0
- **浏览器**: Chromium (Playwright)
- **登录方式**: Cookie 认证
- **API 服务**: ZhipuAI (GLM-Image + GLM-4.6V)

### 关键技术点
1. **标题优化**: 自动移除 EMOJI，防止 CSDN 解析为 `[特殊字符]`
2. **标签限制**: 自动截断至 7 个以内 (CSDN 平台限制)
3. **封面图生成**: 使用 GLM-Image AI 生成个性化封面
4. **强制点击**: 绕过元素遮挡问题
5. **抗干扰机制**: 自动切换焦点，防止新标签页干扰
6. **内容验证**: 粘贴后自动验证完整性 (>90%)

### 遇到的挑战与解决
1. **专栏选择超时**: 被标签输入框遮挡 → 跳过专栏选择，不影响发布
2. **上传按钮遮挡**: 普通点击失败 → 使用 JavaScript 强制点击
3. **文件选择器捕获**: 使用 `expect_file_chooser()` 异步等待
