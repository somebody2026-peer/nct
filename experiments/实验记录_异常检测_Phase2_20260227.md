## 🔍 失败原因深度分析

### 1. 阈值校准策略缺陷

**中位数 +1.5σ 的问题**:
```python
# PE 分布极度右偏
median(PE) ≈ 4000
std(PE) ≈ 6000
threshold = median + 1.5*std = 13000  ← 远高于实际异常值
```

**Φ值的双向困境**:
```python
# Φ值本身已经很小 (0.0003)
median(Φ) ≈ 0.0002
std(Φ) ≈ 0.0001
threshold = median - 1.5*std = 0.00005  ← 接近于 0，无法触发
```

---

### 2. 异常类型难度梯度

| 异常类型 | 视觉变化程度 | 语义变化程度 | 检测难度 |
|---------|-------------|-------------|---------|
| 噪声 | 小 | 小 | ⭐⭐⭐⭐⭐ 极难 |
| 旋转 | 中 | 无 | ⭐⭐⭐⭐⭐ 极难 |
| 遮挡 | 中 | 中 | ⭐⭐⭐ 中等 ✅ |
| OOD | 大 | 大 | ⭐⭐⭐⭐⭐ 极难 |

**洞察**: 只有"语义变化"才能被检测到，纯视觉变化无效

---

### 3. 模型表征特性

**假设**: NCT 学习到的是**高级语义特征**而非低级像素特征

**证据**:
- 对遮挡敏感（破坏语义）
- 对噪声/旋转不敏感（语义完整）
- 对 OOD 不敏感（强行解释为某种数字）

## 🎯 下一步改进方案

### 方案 A: 基于训练数据真实范围（优先级：高）

**原理**: 直接使用训练时的最大/最小值作为基准

**实施**:
```python
# 修改 calibrate() 方法
self.pe_threshold = np.percentile(all_pes, 95) * 1.5  # 95 分位放宽
self.phi_threshold = np.percentile(all_phis, 5) * 0.5  # 5 分位收紧

# 或使用绝对阈值
self.pe_threshold = max(all_pes) * 1.2  # 比最大值高 20%
self.phi_threshold = min(all_phis) * 0.8  # 比最小值低 20%
```

**预期效果**: PE F1 → 0.3-0.5

---

### 方案 B: 监督式阈值调优（优先级：最高）⭐

**原理**: 使用带标签的验证集网格搜索最优阈值

**实施步骤**:

1. **准备验证集**:
   - 200 正常样本 + 100 异常样本（每种类型）
   - 总共 600 样本

2. **网格搜索**:
```python
best_f1 = 0
for pe_thresh in range(3000, 15000, 500):
    for phi_thresh in np.arange(0.0001, 0.0005, 0.00005):
        f1 = compute_f1_on_val(pe_thresh, phi_thresh)
        if f1 > best_f1:
            best_pe, best_phi = pe_thresh, phi_thresh
```

3. **在测试集验证**:
   - 使用独立的 100 正常 +100 异常样本

**预期效果**: Combined F1 → 0.5-0.7

---

### 方案 C: 针对每种异常单独校准（优先级：中）

**原理**: 不同异常需要不同的阈值

**实施**:
```python
# 为每种异常生成校准数据
calibration_data = {
    'noise': generate_noise_anomalies(n=50),
    'rotation': generate_rotation_anomalies(n=50),
    'occlusion': generate_occlusion_anomalies(n=50),
    'ood': generate_ood_anomalies(n=50)
}

# 分别计算 ROC 曲线，找到最佳工作点
for anomaly_type, data in calibration_data.items():
    thresholds[anomaly_type] = find_optimal_threshold(data)
```

**预期效果**: 各异常类型 F1 差异缩小

---

### 方案 D: 引入新的异常信号（优先级：低）

**创新思路**:

1. **重构误差** (Reconstruction Error):
   - 添加 Decoder 模块
   - 计算输入与重构的差异

2. **注意力分散度**:
   - 计算注意力熵的标准差
   - 异常时应该更分散

3. **多候选一致性**:
   - 计算 15 个候选的方差
   - 异常时应该分歧更大

4. **预测层级误差**:
   - 提取 3 层 Predictive Hierarchy 的 PE
   - 逐层检测异常

**预期效果**: 长期研究方向

## 📝 关键发现总结

### ✅ 成功方面
1. **分类任务**: 99.61% 准确率，刷新博客记录
2. **Φ值涌现**: 成功观测到三阶段动力学
3. **PE 正则化**: 验证了 PE 增长与准确率同步的现象
4. **GPU 加速**: 训练效率提升 15-20 倍

### ❌ 失败方面
1. **异常检测**: 所有方法 F1 < 0.15，基本失效
2. **阈值校准**: 中位数策略不适合 NCT 输出分布
3. **无监督方法**: 难以适应复杂异常场景

### 💡 重要洞察
1. NCT 学习到的是**语义级表征**而非像素级特征
2. 只有破坏语义的异常（遮挡）才能被检测
3. 纯视觉扰动（噪声、旋转）被视为"合法变异"
