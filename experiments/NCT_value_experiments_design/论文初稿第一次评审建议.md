**现状**：

> "如果最好的模型只能解释 10% 方差，剩余 90% 来自哪里？这还是真实效应吗？"

**可能原因**：
1. **测量噪声**：单次 γ-cycle 的指标波动较大
2. **个体差异**：不同 MNIST 样本的响应差异
3. **缺失变量**：噪声不是唯一驱动因素

**优化建议**：

**方案 A：坦诚讨论局限性**
```markdown
### 4.5.1 解释方差的考量

虽然二次模型 R² = 0.097 低于传统回归标准，但需考虑：

1. **意识指标的内在变异性**：
   - 单次 γ-cycle 输出受随机初始化影响
   - 个体样本差异（不同数字、书写风格）
   - 类比：人类 EEG 信号 trial-to-trial 变异系数常达 30-50%

2. **互信息的互补证据**：
   - NMI = 0.605 > 0.5 表明强依赖
   - R² 衡量线性预测力，NMI 衡量一般依赖关系
   - 低 R² + 高 NMI = 非线性但真实的关系

3. **效应量 vs 解释方差**：
   - Cohen's d = 0.83（大效应）表明组间差异显著
   - R² 和 d 关注不同问题（预测 vs 差异检测）
```

```python
# 补充分析
def compute_standardized_effect():
    low_noise_data = get_data(noise < 0.3)
    high_noise_data = get_data(noise > 2.0)
    
    # Cohen's d
    pooled_std = np.sqrt(
        (np.std(low_noise_data)**2 + np.std(high_noise_data)**2) / 2
    )
    d = (np.mean(high_noise_data) - np.mean(low_noise_data)) / pooled_std
    
    # 分布重叠度
    overlap = 2 * np.mean(low_noise_data) / (np.mean(low_noise_data) + np.mean(high_noise_data))
    
    return d, overlap

# 如果 d > 0.8，即使绝对变化小，统计上仍有意义
```

### 四、方法学补充

**1. Permutation Test（验证曲线形状显著性）**
```python
def permutation_test_quadratic(n_permutations=10000):
    """检验二次项系数是否显著异于零"""
    observed_a = fit_quadratic(noise_levels, fe_means)['a']
    
    null_a_distribution = []
    for _ in range(n_permutations):
        shuffled_fe = np.random.permutation(fe_means)
        a = fit_quadratic(noise_levels, shuffled_fe)['a']
        null_a_distribution.append(a)
    
    p_value = np.mean(np.abs(null_a_distribution) >= np.abs(observed_a))
    return p_value

# 如果 p < 0.05，证明倒 U 型不是偶然
```

**2. Bootstrap 置信区间**
```python
def bootstrap_vertex_ci(n_bootstrap=5000):
    """估计顶点位置的置信区间"""
    vertices = []
    
    for _ in range(n_bootstrap):
        # 重采样
        indices = np.random.choice(len(fe_means), size=len(fe_means), replace=True)
        boot_noise = noise_levels[indices]
        boot_fe = fe_means[indices]
        
        # 拟合
        params = fit_quadratic(boot_noise, boot_fe)
        vertices.append(params['vertex_x'])
    
    ci_low, ci_high = np.percentile(vertices, [2.5, 97.5])
    return ci_low, ci_high

# 报告：顶点 = 1.028 [0.856, 1.201] 95% CI
```

**3. AIC/BIC 模型比较**
```python
def compute_aic(n, rss, k):
    """AIC = n * ln(RSS/n) + 2k"""
    return n * np.log(rss / n) + 2 * k

n = len(fe_means)
```
